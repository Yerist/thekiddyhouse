# The Kiddy House

Landing page para **The Kiddy House**, una guardería infantil con ambiente cálido y divertido. Incluye secciones de presentación, horarios, galería y contacto.

## Demo

[Ver sitio en vivo](https://yerist.github.io/thekiddyhouse/)

## Tecnologías

- HTML5
- CSS3 (animaciones y diseño responsivo)
- JavaScript (vanilla)

## Correr localmente

No requiere dependencias ni herramientas de build.

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Yerist/thekiddyhouse.git
   ```
2. Abre `index.html` directamente en tu navegador, o usa una extensión como **Live Server** en VS Code para recarga automática.
